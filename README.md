# BXM
Blind XSS Manager, That keeps all the blind xss payloads to use directly from browser and keeps track of the injected URL.

## Features
**Encode** : User can encode payloads directly from the extension.  
**Inject Place** : It remembers the exact URl where you inject payload.  
**Payloads** : It have most updated blind xss paylaods with polyglots.

## Usage

1. Clone
```
git clone https://github.com/SeifElsallamy/Blind-XSS-Manager.git
```
2. Open chrome browser:
3. Go to `chrome://extensions/` and check the box for Developer mode in the top right.
4. Click load unpacked and choose the folder where you clone the repositry
